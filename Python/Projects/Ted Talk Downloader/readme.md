# Download a Ted Talk
A script that will help you to download a Ted Talk video in mp4 format. When you run this script then the ted talk video of the provided url will be downloaded in the same folder as the script in mp4 format.

# How To Use
Eg:
Put this script in the folder and the provide the url like following, 
then run the script with the command ``` python Ted-Talk-Downloader.py https://www.ted.com/talks/devin_shuman_the_genes_you_don_t_get_from_your_parents_but_can_t_live_without```
